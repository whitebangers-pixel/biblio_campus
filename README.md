# BiblioCampus

Application Python de gestion de bibliothèque universitaire.

## Fonctionnalités
- Ajouter un livre
- Lister les livres
- Emprunter un livre
- Rendre un livre
- Afficher des statistiques

## Installation
```bash
git clone https://github.com/whitebangers-pixel/biblio_campus.git
cd biblio_campus
python main.py
```
